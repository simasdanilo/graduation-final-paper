Padrão e consistência

Um aplicativo consistente implementa padrões e paradigmas familiares usando elementos de interface fornecidos pelo sistema, ícones bem conhecidos, estilos de texto padrão e terminologia uniforme. Torne seu aplicativo mais inteligente ao aproveitar padrões visuais e memórias estabelecidos pela plataforma.

Sempre que possível, use controles de navegação padrão uma vez que os usuários já estão familiarizados com esses controles e sabem intuitivamente como interagir com eles.

O mesmo aplicativo nas plataformas Android e iOS mantêm os ícones, a organização e os padrões das suas respectivas plataformas.

Para facilitar a aprendizagem e estimular a memória de curto prazo do usuário o aplicativo deve manter os componentes no mesmo padrão visual e local ao longo de toda a interação. Funcionalidades semelhantes devem ser realizadas por interações similares. A metáfora de cada componente ou recurso deve ser única em toda a aplicação para evitar mal-entendidos.

Gestos

Como regra geral, use os gestos padrões da plataforma. As pessoas estão familiarizadas com os gestos padrões e não apreciam ser obrigadas a aprender maneiras diferentes de fazer a mesma tarefa.
