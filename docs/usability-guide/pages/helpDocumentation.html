Ajuda e documentação

Ajude os usuários a obter mais valor do seu aplicativo apresentando-os para recursos e funcionalidades em momentos contextualmente relevantes, ou seja, centrado na tarefa atual do usuário.

Uma sobreposição apontando as principais características e como interagir com o aplicativo permite aos usuários que entram na sua aplicação pela primeira vez se familiarizar com a interface do seu aplicativo.

Ajude os usuários a descobrir os recursos que sua aplicação oferece. Aborde os usuários para tentar recursos ou funcionalidades novas ou desconhecidas.

Faça a aprendizagem divertida e descoberta. Aprender fazendo é muito mais divertido e eficaz do que ler uma lista de instruções. Use animação e interatividade para ensinar de forma gradual e em contexto. Evite exibir capturas de tela que parecem interativas.

Texto explicativo

Para abordagens menores e mais simples que solicitem a interação do usuário, use uma sugestão de texto.

Um texto explica o que o botão adicionar lhe permite fazer.

Fluxos guiados

Os fluxos guiados passam pelos usuários através de tarefas que exigem várias etapas para serem concluídas.

Um aplicativo de viagens ensina como criar uma viagem passo-a-passo através do fluxo guiado para seus usuários iniciantes.

Respeite o essencial em tutoriais. É bom fornecer orientação para iniciantes, mas a educação não é um substituto para um excelente design. Em primeiro lugar, faça o seu aplicativo intuitivo. Se for necessária muita orientação, revise o design do seu aplicativo.

Qualquer tipo de sistema de ajuda também deve ser exibido a qualquer momento quando solicitado pelo usuário.

Ensinando gestos

Uma vez que os gestos representam uma interação implícita do seu aplicativo é importante ensinar o que eles irão fazer em um texto curto e claro. Você também pode utilizar de animações para ajudar o usuário a entender o gesto.

Um card com animação ensina aos usuários iniciantes que é possível arquivar mensagens deslizando-as para o lado.

Experiência de primeira vez

Forneça uma experiência única para novos usuários de acordo com o tipo de sua aplicação.

Principais Benefícios do Usuário

Destaque em poucas páginas e de forma breve os principais benefícios de usar o seu aplicativo.

Uma apresentação de slides ilustra os principais benefícios e funcionalidades do aplicativo.

Seleção automática

Permita aos usuários personalizarem suas experiências.

Uma tela de seleção permite o usuário personalizar sua experiência selecionando os esportes de seu interesse.

Início rápido

Inicie o usuário diretamente no aplicativo e mostra pequenas dicas ou ajudas na tela.

Um aplicativo de receitas ajuda um usuário iniciante a adicionar sua primeira receita.
