LiloImageButton

public class LiloImageButton
extends AppCompatImageButton

An AppCompatImageButton that ensures that its size is always at least the size of the minimum touch target area defined by the component. The minimum touch target area can be modified by overriding the methods below.

Public methods
int getMinTouchTargetWidth()

Returns the minimum touch target width, in density-independent pixel (dp).

int getMinTouchTargetHeight()

Returns the minimum touch target height, in density-independent pixel (dp).
